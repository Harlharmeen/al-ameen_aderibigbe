🎮 How to Play the Simon Game

Welcome to the Simon Memory Game! Test your memory and see how many levels you can complete.

🧠 Objective

Repeat the sequence of flashing colors for as long as you can. Each round, the game adds a new color to the sequence. Your goal is to remember and repeat the growing pattern.

🕹️ How to Start:

On desktop: Press any key.
On mobile: Tap anywhere on the screen.

🎯 How to Play:

  • Watch carefully: A color will flash — this is the first in the sequence.
  • Click/Tap the color to repeat it.
  • The game will then flash the full sequence again with an added color.
  • Repeat the full pattern** in order each time.
  • Keep going — the sequence gets longer each round.

❌ Game Over:

If you tap the wrong color, the game ends.
The screen will flash and say: "Game Over".
Press any key or tap the screen again to restart.

📝 Tips:

Play with sound on — each color has a unique tone.
Focus and try to visualize or say the pattern aloud.
